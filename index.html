Communal To-Do App

A solution to organize tasks in a busy community.

January 2025 - April 2025
School Project
Flutter, Supabase

Project Overview

For a major third-year subject we had to create a mobile application in order to pass. What I came up with was an app that functions as a to-do list that other people could update which could be helpful for when a group needs to be informed of the same things such as schoolwork and chores. Each to-do list functions in groups that can invite other users to it. Each task can have additional information other than the name of the task and the deadline. It can have assignees and also include additional notes and pictures.

I have passed the subject. Currently working towards deploying an official build for it.

E-Learning System with Collaborative Tools

A system to facilitate online classroom collaboration.

Jan 2025 - Present
Capstone
Flutter, Supabase, Vue.js

Project Overview

With group and direct messaging, file storage, announcements, and a schoolwork system to set-up and organize assignments and projects among students. It is a cross-platform app that focuses to keep an e-learning environment in a single space rather than having to rely on multiple external applications such as Facebook and Messenger for communication and Google Classroom for assignments.

Currently still in development.
